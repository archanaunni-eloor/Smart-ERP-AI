An advanced ERP solution designed for small to medium enterprises, featuring a **State-of-the-art AI Agent** for inventory insights, salary predictions, and automated business reporting.

---

## 🌟 Key Features

- **AI Insights Agent:** Integrated Gemini-powered agent that analyzes inventory and salary data to provide actionable business insights in multiple languages (English & Malayalam).
- **Automated Salary Management:** Seamless salary slip generation with a built-in PDF download feature.
- **Predictive Analytics:** Forecasting sales trends and inventory requirements using Machine Learning logic.
- **Professional Reporting:** Generate "As-it-is" AI reports with a dedicated PDF export functionality.
- **Secure Architecture:** Built with Django, ensuring robust data security and scalable database management.
